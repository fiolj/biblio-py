Biblio-Py
Python Bibliography manager and utilities

Description

Biblio-py is a package to manage bibliography written in python. Currently it contains two subpackages:

  • yapbib: Yet Another Python BIBliography manager tool, mainly for Bibtex files.
  • query_ads: A simple python tool that permits to query Harvard Database

This is the first public release. It works well but I will be glad to hear comments.

Features

  • Open Source software, License GPL (free to use-modify-redistribute)
  • Object oriented model.
  • Parsing of BibTeX and ADS portable format
  • Built in conversion to BibTex, LaTeX, HTML and XML formats.
  • Native Format for store uses standard python pickle module
  • Convert accents and some non-english characters to XML entities.
  • Handles some (little) math, currently superscripts and subscripts. It should not be very difficult to add more.
  • Include scripts to manipulate databases, get bibliography from ADS online database and extract citations from a LaTeX paper
  • You can import the module from your script and perform Python-powered manipulation on the data.

Installation

Currently we provide installation from source and the following ebuild for Gentoo Linux

Manual Installation

  1. Download the compressed archive with the latest release and copy to a temporary directory
  2. Unpack the file and install using the setup 
    
  $>tar -xvzf biblio-py-x.x.x.tar.gz
  $>cd biblio-py-x.x.x
  $>./setup.py install

Gentoo Installation

  1. Download the ebuild and copy it in your local portage tree.
  2. Then run (if you are using standard locations)
    3. 
  $>echo dev-python/biblio-py >> /etc/portage/package.keywords
  $>ebuild biblio-py-x.x.x.ebuild digest
  $>emerge biblio-py

Usage

The program currently includes three scripts: get_papers.py, bibmanage.py and bibextract.py To get help type one of the following: 

  <script> -h
  <script> --help

bibmanage examples

  1. Example: Import the BibTeX file myrefs.bib and save them to the database myrefs.dmp: 
    
  $>bibmanage.py myrefs.bib -d myrefs.dmp
  2. Example: Merge two bibtex files, create a new bibtex file and also dump to database.dmp 
    
  $>bibmanage.py myrefs1.bib  myrefs1.bib -f bib -o bib allrefs.bib -o database.dump  --keep-keys
  $>bibmanage.py myrefs1.bib  myrefs1.bib -f bib -o allrefs.bib -o database.dump
    The first line won't modify the keys. The second (default) will uniformize the keys to the current scheme, which for articles is:
    First seven charachters of name + year + journal abbreviation + p + page
  3. Example: Select only those items with some substring in the key (output to stdout) from a bz2-comprised bibtex file: 
    
  $>bibmanage.py myrefs.bib.bz2 -s substring:key -o -
  4. Example: Select only those items whose author is (among others) autor1,B. and print them in HTML format to file autor1.html use one of the (short or long options are equivalent): 
    
  $>bibmanage.py myrefs.dmp -s autor1,B:author -o autor1.html -f html
  $>bibmanage.py myrefs.dmp --search=autor1,B:author --output=autor1.html --format=html
  5. The last example may be repeated using as source the original BibTeX file, but the parsing is slower. 
    
  $>bibmanage.py myrefs.bib -s autor1,B:author -o autor1.html -f html
  6. Example: Select only those items whose author is (among others) autor1,B. but not autor2,C and print them in LaTeX format to file ejempl.tex: 
    
  $>bibmanage.py myrefs.dmp -s autor1,B:author -x autor2,C:author -o ejemplo.tex -f
  7. Restrict the last example to publications in years between 2004 and 2006 
    
  $>bibmanage.py myrefs.dmp -x autor1,B:author -x autor2,C:author \
                 --start-year=2004 --end-year=2006 -o ejemplo.tex -f

get_papers examples

  1. Example: Get all the papers in the database for one author in a given year 
    
  $>get_papers.py --author=Einstein,A. --year=1905
    you will get BibTeX entries in the standard output. There are options that change the behavior.
  2. Example: Use of other options 
    
  $>get_papers.py --author=Einstein,A. --year=1905 -f html -o biblio.html
  $>get_papers.py --author=Einstein,A. --year=1905 -f latex --output=biblio.tex
  $>get_papers.py --author=Einstein,A. --year=1905 -f latex --sort=date,reverse

bibextract examples

  1. Example: Simplest use is to get the bibliography cited in document.tex (directly or from document.aux). Use one of the following: 
    
  $>bibextract document.tex
  $>bibextract document.aux
    This will create a file: document.bib
